补充

编译debug版本的使用：swift build，可执行文件被放在.build/debug/隐藏目录下；
编译release版本的使用：swift build -c release,可执行文件被放到.build/release/路径下；
清理编译临时文件，删除.build，使用：swift build –clean
清理.build目录和Packages目录，使用：swift build –clean=dist
